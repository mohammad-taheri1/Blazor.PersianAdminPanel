A Blazor web application for Persian (Farsi) Admin Panel

### Required Entities:

- Users `(IdentityUser)`
- Roles `(IdentityRole)`
- UserRoles (Many-To-Many) `(IdentityUserRoles)`
- Courses
- CourseLessons
- Orders
- Transactions
- Balances
- Tickets
