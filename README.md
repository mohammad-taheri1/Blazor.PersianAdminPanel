A Blazor web application for Persian (Farsi) Admin Panel

### Required Entities:

- Users `(IdentityUser or custom)`
- Roles `(IdentityRole or custom)`
- UserRoles (Many-To-Many) `(IdentityUserRoles or custom)`
- Courses
- CourseLessons
- Orders
- Transactions
- Balances
- Tickets
